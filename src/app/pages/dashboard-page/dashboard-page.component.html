web ระบบจัดการข้อมูลโครงการ NUD Prep Test

ประจำปี 2018
ยินดีต้อนรับผู้ประสานงานศูนย์สอบ เผยแพร่เมื่อ: 14 กันยายน 2561 เวลา 12.30 น.

ยินดีต้อนรับผู้ประสานงานศูนย์สอบเข้าสู่ระบบจัดการข้อมูลโครงการ NUD Prep Test ประจำปี 2018 ระบบนี้เป็นระบบที่จะใช้บันทึกข้อมูลศูนย์สอบ และผู้สมัครเข้าร่วมทดสอบ ตลอดจนการดำเนินการต่าง ๆ ที่เกี่ยวข้องกับโครงการฯ ในเบื้องต้นนี้ขอให้ผู้ประสานงานศูนย์สอบดาวน์โหลดคู่มือการบันทึกข้อมูลศูนย์สอบที่เมนูด้านบน และดำเนินการตรวจสอบ หรือแก้ไขข้อมูลศูนย์สอบให้ถูกต้อง ในส่วนการบันทึกข้อมูลสถานที่ของศูนย์สอบ หรือการบันทึกข้อมูลผู้สมัครเข้าร่วมทดสอบนั้น ระบบจะเปิดให้ดำเนินการในวันที่ 17 กันยายน 2561 หากพบว่าระบบขัดข้องโปรดติอต่อเจ้าหน้าที่หมายเลขโทรศัพท์ 081-675-2065 เพื่อให้คำแนะนำ และแก้ไขปัญหาการใช้งานระบบของท่าน และหากโครงการมีเรื่องแจ้งให้ศูนย์สอบทราบจะประชาสัมพันธ์ไว้ที่ระบบนี้ โปรดเข้าสู่ระบบเพื่อติดตามข่าวสาร
